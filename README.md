# Installation

### install on koteli-tui (in terminal)

  - **MacOS/Linux (Recommended):**
    ```bash
    curl -fsSL https://ai.kxki.dev/install.sh | bash
    ```

  - **Windows (Recommended):**
    ```powershell
    irm https://ai.kxki.dev/install.ps1 | iex
    ```

  - **Bun (Recommended):**
    ```bash
    bun add -g @kxki-dev/koteli
    ```

  - **NPM (Deprecated):**
    ```bash
    npm install -g @kxki-dev/koteli
    ```
